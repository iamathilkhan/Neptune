About Neptune

Neptune is a machine learning application that combines neural networks for prediction with semantic search for contextual understanding. It uses:

  • TensorFlow/Keras: Deep learning models for predictions
  • FAISS: Efficient similarity search over 50,000+ ocean records
  • Sentence Transformers: Semantic embeddings for context retrieval
  • Hugging Face Transformers: Language model for natural responses
